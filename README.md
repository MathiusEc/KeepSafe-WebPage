
# Keep Safe - Precision Agriculture Drone Website

## Project Status

**Published and live in production.**
This repository is for documentation and backup purposes only.

## Technologies Used

- Semantic HTML5
- CSS3 (modern animations and effects)
- Vanilla JavaScript
- Mobile-First Responsive Design
- SEO Optimization
- Schema.org (Structured Data)

## Repository Structure

```

├── index.html              # Home page

├── nosotros.html           # About us page
├── contactanos.html        # Contact page
├── 404.html                # Custom error page
├── styles.css              # Main styles
├── animations.js           # Animations and effects

├── sitemap.xml             # Sitemap
├── CNAME                   # Custom domain (client-owned)

```
